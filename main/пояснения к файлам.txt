coord_zone_perf - координты прямоугольников (активных зон перфорации) (x, y);
grid_coord - координаты разбиений на подобласти параллелепипедной сетки (количество линий подобластей по x, координаты линий по x, 
						    			 количество линий подобластей по y, координаты линий по y,
					            			 количество линий подобластей по z, координаты линий по z
                                                                         номер подобласти, первая координата по x, вторая координата по x, первая координата по y, вторая координата по y, первая координата по z, вторая координата по z);
num_splits - количество разбинений расчетной области (x, y, z);
coef - массив значений k, k_ph, eta_ph для каждой подобласти (первая строка для k, 
                                                              вторая строка для k_ph,
                                                              третья строка для eta_ph);

K - массив значений структурной проницаемости для каждой подобласти;
ph - значения k_ph и eta_ph для каждой фазы для каждой подобласти (первое число ph - количество фаз, второе - Ф (коэф. структурной пористости породы),
                                                                   следующие ph чисел - значения динамической вязкости для
                                                                   каждой жидкости (const во всей области),
                                                                   следующие 1, ..., n (n - кол-во подобластей) строк по ph значений -
                                                                   значения множителя структурной проницаемости, зависящие от насыщенности в i-й области,
                                                                   для каждой фазы от 1 до ph);
