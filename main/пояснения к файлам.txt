coord_zone_perf - координты зон перфорации (первое число n - количество зон перфорации;
                                            затем 2n пар чисел x и y, определяющих границы зоны перфорации (две точки на одну зону));
grid_coord - координаты разбиений на подобласти параллелепипедной сетки (количество линий подобластей по x, координаты линий по x, 
						    			                         количество линий подобластей по y, координаты линий по y,
					            			                    количество линий подобластей по z, координаты линий по z
                                                                          номер подобласти, первая координата по x, вторая координата по x, первая координата по y, вторая координата по y, первая координата по z, вторая координата по z);
K - массив значений структурной проницаемости для каждой подобласти;                                                                          
num_splits - количество разбинений расчетной области (первая строка для x, вторая - для y, третья - для z,
                                                      всего в строке m пар чисел для каждой координаты свое (m - реальное количество границ координаты, создающих отдельные подобласти(кол-во линий подобластей + кол-во линий от зон перфорации))
                                                      первое число в паре - кол-во необходимых разбиений данной подобласти; второе - коэф. разрядки для задания неравномерной сетки (1 - равномерная)
                                                      последние 3 числа - создание вложенных сеток по каждой координате. 0 - базовая сетка, 1 - в 2 раза больше разбиений по выбранной координете, 2 - в 4 раза больше, и тд);
ph - значения k_ph и eta_ph для каждой фазы для каждой подобласти (первое число ph - количество фаз, второе - Ф (коэф. структурной пористости породы),
                                                                   следующие ph чисел - значения динамической вязкости для
                                                                   каждой жидкости (const во всей области),
                                                                   следующие 1, ..., n (n - кол-во подобластей) строк по ph значений -
                                                                   значения множителя структурной проницаемости, зависящие от насыщенности в i-й области,
                                                                   для каждой фазы от 1 до ph);
set_flow_zone_perf - задание фиксированного объема закачиваемой/отбираемой жидкости через зону перфорации (всего столько значений, скользо задано зон перфорации)
                                                                                                          (если жидкость закачивается (значение положительное), то дополнительно указываются
                                                                                                           значения множителя структурной проницаемости для каждой фазы от 1 до ph для этой скважины)
